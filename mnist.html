Recognition of numbers using a model trained with MNIST dataset

Source code

MNIST is a widely recognized dataset in the realm of machine learning and computer vision. It serves as a crucial benchmark for evaluating the performance of various algorithms and models. Comprising 70,000 grayscale images of handwritten digits, each represented as a 28x28 pixel grid, MNIST offers an ideal playground for tasks like digit recognition and image classification.

However, it is not compatible with mouse-written numbers, so I did not use it in my project on handwriting recognition.

In conclusion, this project aimed to recognize numbers using a model trained on the MNIST dataset. Two models, KNN and neural network, were tested on handwritten numbers. The KNN model achieved 97.05% accuracy, while the neural network model achieved 97.62% accuracy but misclassified two numbers. Overall, both models demonstrated impressive performance, showcasing their potential for accurate number recognition tasks.
